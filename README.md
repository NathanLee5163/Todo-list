# Todo-List

A simple and minimalistic **desktop Todo List app** built with [Electron](https://www.electronjs.org/).  
Tasks are saved to a local text file, so your notes persist even after closing the app.

---
## How to run
- Clone this repository
- install npm
- install npm i -D electron electronmon
- npm start
- ctrl + C 
## Features
- Add tasks quickly using the text box at the bottom  
- Newest tasks appear at the top  
- Delete tasks with a single click  
- Automatically saves tasks to a text file (`tasks.txt`)  
- Tasks reload automatically when you reopen the app
